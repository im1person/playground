## Hub 節日篩選 / Festival filters

首頁可依節日篩選工具與遊戲，`data-festival` 與篩選值對應如下。

| 篩選值 (data-filter) | 節日 |
|----------------------|------|
| `christmas` | 聖誕節 |
| `lunar-new-year` | 農曆新年 |
| `halloween` | 萬聖節（預留） |
| `mid-autumn` | 中秋節（預留） |
