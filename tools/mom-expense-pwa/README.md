# 屋企數簿（Mom Expense PWA）

家居開支記帳用 **PWA**：單一 `index.html`，**繁體中文介面**，資料只存在本機瀏覽器（`localStorage`），可離線使用並可加到手機主畫面。

## 功能摘要

- **入數**：自選分類、金額（支援簡單四則輸入）、日期；可標示 **已交／未交**，未交時可填 **到期日**，方便追帳單。
- **紀錄**：依月份檢視，按分類群組與小計；可 **刪除** 單筆或 **清空所有紀錄**。
- **未交 → 已交**：每筆未交紀錄可 **✅ 標記已交**；若該筆來自帳單（有 `dueDate`），誤按時可用 **⏪ 撤銷** 還原為未交（一般「立刻付清」的紀錄不顯示撤銷，介面保持簡潔）。
- **設定 — 分類**：新增分類（Emoji + 名稱）、**編輯**（名稱會 **聯動更新** 所有相關紀錄）、刪除；常用 Emoji 快速選擇。
- **PWA**：`manifest.json`、`icon-512.png`，適合安裝成獨立 App。

## 本機資料鍵名

| 鍵名 | 用途 |
|------|------|
| `travel-expense-records` | 開支紀錄（陣列） |
| `expense-categories` | 自訂分類（名稱 + Emoji） |

重新整理或關閉分頁後資料仍保留；清除瀏覽器網站資料會一併刪除。

## 專案檔案

| 檔案 | 說明 |
|------|------|
| `index.html` | 應用程式本體（HTML / CSS / JS） |
| `manifest.json` | Web App Manifest（名稱：`屋企數簿`） |
| `icon-512.png` | 應用程式圖示（實際像素請與 manifest `sizes` 一致） |
| `make_icon_512.py` | （選用）從全畫面 mockup 中央裁切輸出 512×512 的腳本 |
| `mockup.png` | （選用）設計／示意用截圖 |

## 使用方式

1. 以本機或靜態伺服開啟目錄中的 `index.html`（建議用 `http(s)://`，以便 PWA / `manifest` 正常運作）。
2. 在手機瀏覽器選「加入主畫面」或「安裝」即可當 App 使用。

## 技術備註

- **Vanilla JavaScript**，無建置步驟。
- 圖示尺寸若更換，請同步更新 `manifest.json` 的 `sizes` 與 `index.html` 內 `apple-touch-icon` 的 `href`。
