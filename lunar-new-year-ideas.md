## 遊戲 (Games)

### 1. 春聯對對碰 (Spring Couplet Match) — ✅ 已完成

- **概念**：記憶翻牌配對，牌面為春聯吉祥字或成對句子（如「春」「福」「吉祥」等）。
- **實作**：`/games/spring-couplet-match/`，吉祥字 春／福／祿／壽／吉／祥／如／意／龍／鳳／財／安 配對，關卡遞增。
- **核心機制**：
  - 網格翻牌，翻到相同字/句即配對成功。
  - 可選「單字模式」（春、福、祿、壽等）或「短句模式」（四字吉祥話）。
  - 計步數與時間，支援簡單難度（格數、牌面種類）。
- **可選功能**：
  - 多組主題（傳統紅底金字、水墨、Q 版）。
  - 雙語標籤（繁體中文 / 英文吉祥話對照）。
  - 最佳紀錄儲存（localStorage）。

### 2. 紅包接接樂 (Red Envelope Catch)

- **概念**：輕量 arcade，從天而降的紅包與金幣，玩家移動接盤接住紅包得分，避開或區分「空包」。
- **核心機制**：
  - 鍵盤或觸控控制接盤左右移動。
  - 紅包、金幣落下，接到加分；可選「空紅包」接到扣分或扣時間。
  - 簡單關卡：隨時間加快掉落速度或種類增多。
- **可選功能**：
  - 連接到一定數量觸發短暫「金雨」加分。
  - 簡單音效與過年 BGM 開關。
  - 結算畫面：接到數量、總分、最佳紀錄。

### 3. 十二生肖記憶 (Zodiac Memory) — ✅ 已完成

- **概念**：以十二生肖為主題的記憶配對遊戲，可結合當年生肖高亮或「本命年」小提示。
- **實作**：`/games/zodiac-memory/`，十二生肖 emoji 配對，關卡 4～12 對，紅金主題。
- **核心機制**：
  - 生肖圖案或名稱配對（圖–圖、圖–名、名–名可選）。
  - 難度：6 對、8 對、12 對。
  - 顯示農曆新年與當年生肖（如龍年），增加節慶感。
- **可選功能**：
  - 每隻生肖簡短介紹（習俗、性格象徵）。
  - 多語（繁中 / 英文生肖名）。
  - 與「農曆新年倒數」工具同年份連動（顯示當年生肖）。

### 4. 年夜飯擺盤 (Reunion Dinner Platter)

- **概念**：類似聖誕樹裝飾，拖曳年夜飯常見菜色到圓桌或大盤上，自由擺放完成一桌象徵性年夜飯。
- **核心機制**：
  - 從側邊欄拖曳菜餚（魚、雞、年糕、餃子、蘿蔔糕等）到圓桌。
  - 可旋轉、縮放（若實作簡單），放好後顯示菜名與吉祥寓意（年年有餘、步步高升等）。
  - 完成一桌後可截圖或「分享」預覽（下載為圖片）。
- **可選功能**：
  - 多種桌布與盤子樣式。
  - 每道菜短句說明（文化小知識）。
  - 雙語（菜名與寓意繁中 / 英文）。

### 5. 鞭炮連線消除 (Firecracker Match)

- **概念**：三消或連線消除，圖案為新年元素（鞭炮、燈籠、春聯、橘子、金元寶等）。
- **核心機制**：
  - 經典三消：三顆以上同類相鄰可消，計分。
  - 或改為「連線」：相鄰同圖案連成一線即消。
  - 關卡目標：達到分數、消除指定圖案數量、在步數內完成等。
- **可選功能**：
  - 消除時簡單鞭炮/煙火動畫。
  - 難度分級（盤面大小、步數限制）。
  - 無盡模式與最高分紀錄。

### 6. 猜燈謎 (Lantern Riddle Quiz)

- **概念**：簡易猜謎問答，題目為傳統燈謎或新年相關謎語（字謎、物謎、成語謎）。
- **核心機制**：
  - 一題一題呈現謎面，使用者輸入或選擇答案。
  - 答對顯示解析（為何是這個答案），答錯可顯示提示或正解。
  - 題庫可預先寫死 20–30 題，或從 JSON 載入方便擴充。
- **可選功能**：
  - 繁中 / 英文題目與解析（部分謎語可只提供中文）。
  - 難度標籤（易/中/難）與篩選。
  - 答題紀錄與「今日燈謎」每日一題（依日期種子隨機）。

## 工具 (Tools)

### 1. 農曆新年倒數 (Lunar New Year Countdown) — ✅ 已完成

- **概念**：與聖誕倒數類似，倒數至農曆正月初一（可依每年農曆自動計算或預設下一個春節日期）。
- **實作**：`/tools/lunar-new-year-countdown/`，初一／除夕快速設定、時區選擇、當年生肖顯示（2025～2036 對照表）。
- **核心機制**：
  - 顯示距下次農曆新年（初一）的天、時、分、秒。
  - 時區選擇（與現有 Christmas Countdown 一致）。
  - 可顯示當年生肖與簡短吉祥話。
- **可選功能**：
  - 小年夜、除夕、初一、初二等關鍵日倒數或標註。
  - 多語介面（繁中 / 英文）。
  - 與「十二生肖記憶」共用當年生肖資料。

### 2. 紅包計算機 (Red Envelope Calculator) — ✅ 已完成

- **概念**：幫忙算要包多少紅包、怎麼分攤，或反推「總預算下每人包多少」。
- **實作**：`/tools/red-envelope-calculator/`，總預算／人數、吉利金額快捷鍵、對象清單、複製摘要，TWD/HKD/CNY/USD。
- **核心機制**：
  - 輸入：總預算（或「要包給幾人」+「每人希望金額」），可選幣別。
  - 輸出：建議每人金額（可整除、吉利數字優先，如 88、168、600、1200）。
  - 可列「對象」（稱謂）與金額對照表，方便核對。
- **可選功能**：
  - 吉利數字建議（尾數 8、6，避開 4 等）。
  - 常見關係預設（長輩、平輩、晚輩、同事）與參考區間。
  - 匯出為純文字或 Markdown 清單，方便複製。

### 3. 春聯生成器 (Spring Couplet Generator) — ✅ 已完成

- **概念**：網頁版簡易春聯產生器，選版型、輸入文字，產出可下載或列印的春聯圖。
- **實作**：`/tools/spring-couplet-generator/`，完整版（橫批＋上聯＋下聯）或單字（福／春）、預設吉祥話、下載 PNG、列印。
- **核心機制**：
  - 模板：橫批 + 上聯 + 下聯（或單張「福」「春」）。
  - 使用者輸入自訂文字，即時預覽。
  - 字型、底色（紅、金、黑）、邊框樣式可選。
  - 輸出：下載 PNG 或列印友善頁面。
- **可選功能**：
  - 內建常用吉祥話、成語選單（一鍵填入）。
  - 雙語（繁中 / 英文）範例與標籤。
  - 多張春聯排成 A4/Letter 一頁，方便一次列印。

### 4. 年夜飯菜單規劃 (Reunion Dinner Menu Planner)

- **概念**：規劃一頓年夜飯的菜色、人數與份量，產出採買清單與備料提醒。
- **核心機制**：
  - 從預設菜單勾選（魚、雞、豬腳、長年菜、餃子、湯圓等）或自訂菜名。
  - 輸入人數，每道菜可選份量（小/中/大盤或人數倍數）。
  - 產出：菜單列表 + 簡易採買清單（可依「份量」粗估，如「魚 1 尾」「餃子 30 顆」）。
- **可選功能**：
  - 每道菜附吉祥寓意與簡短做法連結或提示。
  - 匯出 Markdown / 純文字，方便列印或分享。
  - 與「年夜飯擺盤」遊戲共用菜名與寓意資料。

### 5. 拜年行程表 (Visiting Schedule)

- **概念**：輕量工具，記錄過年要拜訪的對象、時間與備註（例如帶什麼禮、紅包是否已包）。
- **核心機制**：
  - 列表新增/編輯/刪除：對象稱謂、預計日期、備註。
  - 可勾選「已拜訪」「紅包已備」等狀態。
  - 資料存於 localStorage，離線可用。
- **可選功能**：
  - 與紅包計算機連動：從「要包給誰」清單一鍵帶入拜年對象。
  - 簡單日曆檢視（初一到初五）。
  - 匯出為清單文字檔。

## 技術與資源備註

- **農曆日期**：農曆新年（正月初一）每年國曆日期不同，可用現成 library（如 `lunar-javascript`、`lunar-core`）或預先寫好未來數年的對照表。
- **多語**：與現有 `locale-menu.js`、`data-en` / `data-zh-Hant` 一致，新頁面沿用相同 pattern。
- **主題**：沿用 `theme-toggle.js`、`style.css`、`kawaii-style.css`，新年主題可在該頁加一組紅色/金色 accent 或專用 CSS 變數。
- **圖示**：新遊戲/工具需對應 `assets/images/` 下 SVG（可後補 kawaii 版），並在 `index.html` 新增對應區塊與連結。

此檔案可作為農曆新年專案的總規劃。目前已實作：農曆新年倒數、紅包計算機、十二生肖記憶、春聯對對碰、春聯生成器；其餘項目可依上表優先順序逐步加入。首頁與 README 已加入上述五項之連結與說明。
